# idoHack収益化案
## サブスクリプション
| **暫定内容** | 一般 | 松プラン | 竹プラン | 梅プラン |
| - | - | - | - | - |
| 値段 | 0 | 480 | 980 | 1,480 |
| 有料演習 | 購入すれば可能(700円/1演習) | 全て可能 | 全て可能 | 全て可能 |
| 有料演習の解説 | 購入済み演習のみ<br> | 購入すれば見れる(300円/1演習) | 全て可能 | 全て可能 |
| 演習の事前公開 | しない | しない | 1週間前 | 1か月前 |
| Slack参加 | 不 | 可 | 可 | 可 |
| チームの作成/編集 | 不 | 不 | 不 | 可 |
| バーチャルCTF開催 | 不 | 不 | 不 | 可 |

## スポンサー

| **暫定内容** | ブロンズ | シルバー | ゴールド |
| :-: | :-: | :-: | :-: |
| 料金 | 30,000/月 | 50,000/月 | 100,000/月 |
| スポンサーテキスト | スポンサーページ | スポンサーページ | スポンサーページ |
| スポンサーロゴ | なし | スポンサーページ | スポンサーページ/課題一覧 |
| サブスクアカウントの発行 | 5万円分まで | 8万円分まで | 15万円分まで |

※プランは組み合わせ可能(ブロンズ+ゴールドで20万円分のアカウント発行をするためなど)
　その場合は、大きいプランが適用される(上記の例ならゴールドプランになる)
