# theHarvester
#ペネトレーションテスト #セキュリティ 
## 概要
> theHarvester（ザハーベスター）は、赤チーム評価または侵入テストの調査段階で使用するために設計された、使いやすくても強力なツールです。このツールは、オープンソースインテリジェンス（OSINT）の収集を行い、ドメインの外部脅威の状況を判断するのに役立ちます。ツールは、複数の公開リソースを使用して、名前、メールアドレス、IPアドレス、サブドメイン、およびURLを収集します。これらのリソースには以下が含まれます (引用:GitHub-README.md)

**インストール**
``` bash
git clone https://github.com/laramies/theHarvester.git
python3 -m pip install -r requirements.txt
```
